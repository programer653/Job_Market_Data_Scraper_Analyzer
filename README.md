Job Market Skill Analyzer
Personal Python project that scrapes job postings, cleans each job description, analyzes the skills asked, and visualizes the most needed skills for each roles: such as Software Developer, Web developer ...

----------------------------------------------------------------------------------------------------------------------------------------------------------------
This project demonstrates capabilities in : 
- Web scraping

- Natural language processing

- Data cleaning

- Keyword extraction

- Data visualization

- Building dashboards with Streamlit

---------------------------------------------------------------------------------------------------------------------------------------------------------------

Installation & Setup
1. Clone the repository
git clone <your-repo-url>
cd job-market-analyzer

2. Create a virtual environment
python -m venv venv
source venv/bin/activate   # macOS/Linux
venv\Scripts\activate      # Windows

3. Install dependencies
pip install -r requirements.txt

4. Run App
run scraper.py then cleaner.py then anbalyzer.p;y then visualizer.py.
Then run app.py to see the dashboard
